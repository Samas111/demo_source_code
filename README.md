# Dancefy – Backend Demo (zjednodušená verze)

Tento repozitář obsahuje zjednodušenou ukázku backendové části projektu Dancefy.

Slouží jako demonstrace způsobu, jakým navrhuji backendovou logiku, pracuji s databází a řeším základní bezpečnostní a aplikační principy.

Nejedná se o kompletní produkční aplikaci - část logiky byla zjednodušena a citlivé části byly odstraněny.
